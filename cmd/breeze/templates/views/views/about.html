About

Breeze is a high-performance Go HTTP + WebSocket framework built on gnet.

Template Engine Features

  • SPA navigation — no full-page reloads
  • Views wrap inside a shared layout automatically
  • Components are reusable HTML fragments with data
  • Dev mode for hot-reload during development
  • Template caching in production

How partial rendering works

When the Breeze SPA runtime intercepts a link click it sends the same request with the header X-Breeze-Partial: true. The server detects this and returns only the {{"{{"}}define "content"{{"}}"}} block — no layout, no scripts. The client swaps it into #breeze-app in milliseconds.
